# Hesabpoor, Personal Finance Manager
Hesabpoor is a clean, lightweight personal finance tracking web application that helps you monitor your income and expenses easily. It focuses on simplicity and uses basic web technologies, offering essential budgeting tools without depending on outside APIs or templates.

## Tech Stack:
- Backend: Django (Python)
- Frontend: Vanilla HTML, CSS, and JavaScript

## Key Features:

- User Registration & Authentication: Secure user accounts with personal profiles.
- Transaction Management: Record your financial transactions (income and expenses) easily.
- Categorization: Organize transactions with custom categories for better insight.

## Financial Overview Dashboards:
- View your total balance.
- Track monthly income and expenses with clear summaries.
- Monitor your net flow for the current month.
- Transaction History: A fast, searchable, and sortable ledger for quick access to all your past transactions.
- Customizable User Profile: Personalize your account settings.

## Project Philosophy:
This project is built from scratch using basic web technologies. It serves as a practical example of a full-stack application, showing how Django handles data models, authentication, and server-side logic. At the same time, a vanilla HTML/CSS/JS frontend creates a responsive and interactive user interface without external frontend frameworks or financial data APIs.

## Ideal For:
Users looking for a simple, self-contained financial ledger and developers interested in a clean Django and vanilla JS full-stack example.
